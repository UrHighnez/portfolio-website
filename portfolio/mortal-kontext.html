Mortal Kontext

The DIY game

One of my earlier Unity projects: a 2D fighting game built from scratch. Only the animations come from Mixamo (plus the occasional hand from friends) — everything else is self-made.

It plays with keyboard or game controllers via Unity's input system. A personal project rather than a study one — though I did use the university's GitLab for version control.
