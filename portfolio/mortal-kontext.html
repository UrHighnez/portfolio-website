Mortal Kontext

The DIY-Game

One of my earlier Unity-Projects was an attempt at creating a 2D fighting game from scratch. I took only the animations from Mixamo and got the occasional help from some friends. Everything else is self-made.

The game can be played with keyboard as well as game controllers through the input system by Unity. I should mention that, although I used my university's GitLab as a Version Control, the project itself was not study related.
